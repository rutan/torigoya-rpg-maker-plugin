# Torigoya RPG Maker Plugins

> RPG ツクール用の JavaScript プラグイン

- website
  - https://torigoya-plugin.rutan.dev
- release files
  - https://github.com/rutan/torigoya-rpg-maker-plugin/tree/gh-pages

## Setup

```bash
$ corepack enable pnpm
$ pnpm install
```

## Task

```bash
# build
$ pnpm run build
$ pnpm run copy

# test
$ pnpm run test

# create new plugin
$ pnpm run scaffold
```
